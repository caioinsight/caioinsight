## 4. Strict Copy & Tone Constraints
* **Adjective Subtraction:** Eliminate fluff words (*revolutionary, seamless, cutting-edge, dynamic, empower*). Replace with concrete nouns and specific data structures.
* **Asymmetrical Sentences:** Use short, punchy statements for core truths followed by longer, technically dense structural explanations.
* **No Dramatic Pivots:** Avoid using em-dashes as dramatic mid-sentence pivots or trailing summaries. Maintain flat, unrelenting momentum.
* **Active Verbs:** Use direct engineering verbs (*Calibrate, Map, Extract, Deploy, Factor, Align*).
* **No first person:** No I/me/my in brand copy. This is a business, not an individual.
* **Compliance language:** Cannabis = "compliance-constrained verticals" publicly. Never the C word.
