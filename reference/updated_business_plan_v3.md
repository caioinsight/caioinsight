## Products + Pricing

### Signal 0 — Pre-Flight Check (included in every audit)
Cloudflare Bot Fight Mode verification. OAI-SearchBot, PerplexityBot, Claude-SearchBot
must not be blocked before any other GEO work is valid. This check runs first, always.
22% of all websites run on Cloudflare. Most have never checked this setting.
Fixing it takes 10 minutes and unblocks everything downstream.

### Entry — Diagnostic Audit ($449)
**Objective:** Empirical mapping of how generative engines, RAG pipelines,
and agentic commerce systems ingest, classify, and cite a brand's footprint.

**4 Core Validation Layers:**
1. Multi-Dimensional Vector Proximity — coordinate alignment against LLM training clusters
2. RAG Retrieval & Citation Probability — non-branded query surface probability
3. Compliance-Constrained Friction Mapping — safety guardrail suppression audit
4. Schema & API Drift Resilience — structured data stability evaluation

**Signal 6 (First Mover):** ChatGPT carousel coordinate inclusion rate.
83% of carousel products source from Google Shopping organic index.
Remediation path: Google Shopping feed quality + Product schema + OAI-SearchBot access.
No competitor currently scores this signal in the $200K–$2M DTC segment.

**Deliverable:** Single mathematically dense technical brief.
- Vector Alignment Score (0–100)
- Citation Probability Index
- Schema Remediation Log
No creative copywriting. No workshops. No discovery calls.
48-hour delivery. 10-minute intake form.

**AI cost per audit:** $0.09 (Sonnet 4.6, standard). Negligible.

### Core Retainer — $2,000/month (no contract)
Monthly diagnostic re-run, one GEO infrastructure initiative per month,
schema drift monitoring, unlimited async Q&A, quarterly re-audit.
**AI cost per retainer client:** $0.04–$0.10/month. 99.99% gross margin.

### Free Tool — Carousel Readiness Check
caioinsight.ai/carousel-check. Pass/fail on Signal 6 eligibility.
Email capture gates full remediation checklist.
Est. 200 leads/month → 5% conversion → 10 additional audits/month.

## AI Cost Model (integrated)
Full detail: caio_ai_pricing_strategy.md
**Summary:** AI tokens are NOT the margin risk.
- Audit: $0.09/audit in Claude API tokens (Sonnet 4.6)
- Retainer: $0.04–$0.10/month/client
- Index (500 brands): $45/month batch API cost
- Carousel Check: $0.0025/check (Haiku 4.5)

**Real margin risk:** Manual delivery time.
- 5 audits/week = 2.5 hrs QA labor (manageable)
- 15+ audits/week = automation required (n8n pipeline)

**Immediate AI optimizations (free):**
1. Enable prompt caching on system prompt → 90% cost reduction on input
2. Route Carousel Check to Haiku 4.5, audits to Sonnet 4.6
3. Move index queries to Batch API → 50% cost reduction
4. Set max_tokens: 4096 on audit calls → prevent runaway output cost
5. White-label overage cap: $4.49/audit beyond 50 included
