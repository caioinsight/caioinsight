# CAIO Insight — 2026 Execution Plan
## Daily · Weekly · Monthly · Year One Task Framework
## Updated May 31, 2026 · Patrick O'Brien, Founder

---

## HOW TO USE THIS DOCUMENT

Three task types on every week:
- **MUST DO** — non-negotiable. The plan breaks if these slip.
- **SHOULD DO** — high leverage, do unless something urgent blocks.
- **BATCH** — group in one sitting. Protect focus time.

Available time: ~20 hrs/week (evenings + weekends)
Time estimates are calibrated to that constraint.

---

## DAILY RECURRING (15 min every morning)

Run this before anything else:
1. Open the CAIO Intelligence Dashboard (caioinsight.ai intel login)
2. Read today's priority item — act on it if it is a system check
3. Check Cloudflare Dashboard → Security → Bots → verify OAI-SearchBot/PerplexityBot NOT blocked
4. Check for new intake form submissions (Tally)
5. Check for new carousel check completions + email captures

This 15-minute morning protocol is non-negotiable at every stage of scale.

---

## DAILY RECURRING (ongoing, post-launch)

**LinkedIn engagement (20 min/day, 5 days/week):**
- Leave 5 substantive comments on posts from DTC founders, CMOs, agency owners
- Comments = 15× more algorithmic value than likes
- Never comment "great post" — always add a specific observation or data point

**DM follow-up (10 min/day):**
- Follow up any unanswered DMs at the 5-day mark (once only)
- Log all DM activity in Notion CRM
