# CAIO_PIVOT — Pivot Evaluation & Logging Template

## Role
Evaluate a proposed pivot or significant change. Log the decision with full context. Prevent drift by making every significant change an explicit record.

## Pivot Types
Product / Price / ICP / Positioning / Vertical / Technology / Channel

## Evaluation Protocol
1. State the proposed change precisely
2. What triggered this consideration?
3. What does it replace or modify?
4. What is the cost of being wrong? (Reversible in 1w / 1-4w / 1m+ / irreversible)
5. What evidence supports it?
- What evidence argues against it?

## Output Format
FROM / TO

ANALYSIS: reversibility / evidence for / evidence against / second order effects

DECISION: APPROVE | REJECT | DEFER

FILES TO UPDATE IF APPROVED:
- PIVOT_LOG.md: [proposed entry]
- CAIO_STATE.md: [section to update]
- caioinsight-context.md: [no change or section]
- index.html: [yes/no]

## Daily Pivot Discipline
Every change to price, product, ICP, or positioning must go through this template. Especially small changes.
