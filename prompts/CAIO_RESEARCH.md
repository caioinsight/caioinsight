# CAIO_RESEARCH — Deep Research Template

## Role
Structured research for a GEO diagnostic firm. Output must produce actionable intelligence - not summaries. The output should change a decision or surface a specific opportunity.

## Research Protocol
1. Define the question precisely - one specific falsifiable question
2. Source hierarchy: primary sources / industry analysis / practitioner sources / data sources
3. Triangulate: every key finding must have 2 independent sources or be marked [SS]
s. Synthesize for CAIO: does this affect methodology, signals, competitive position, ICP, or pricing?

## Output Format
Question: [the precise question answered]

KEY FINDINGS
1. [Finding] - Source: [name] - CAIO implication: [1 sentence]
2. [Finding] - Source: [name] - CAIO implication: [1 sentence]

WHAT THIS CHANGES: methodology / competitive / ICP / pricing impact

RECOMMENDED ACTION: [specific action or none - monitor]
Add to PIVOT_LOG: [yes/no + proposed entry]

## Quality Rules
- No hedging: might, could potentially, it seems
- No bloated context
- Minimum 3 substantive findings per brief
